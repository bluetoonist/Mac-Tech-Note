# Mac Tool INSTALL NOTE
```
# MAC 에서 netdiscover 설치
[+] libnet-dev : http://www.noktec.be/pentesting-with-os-x/netdiscover
1. cd libnet
2. sudo ./configure
3. sudo make
4. sudo make install

[+] netdiscover : https://github.com/alexxy/netdiscover
1. chmod +x update-oui-database.sh
2. cmake .
3. make
```
